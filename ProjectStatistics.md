\# 14. Ümumi Qiymətləndirmə



AzAgroPOS çoxqatlı arxitekturaya malik, peşəkar səviyyədə hazırlanmış Windows Forms əsaslı ERP/POS sistemidir.



Layihədə:



\- Repository Pattern tətbiq edilmişdir.

\- MVP (Model-View-Presenter) arxitekturasından istifadə olunur.

\- Dependency Injection tətbiq edilmişdir.

\- Entity Framework Core vasitəsilə Code First yanaşması istifadə olunur.

\- Asinxron proqramlaşdırma (`async/await`) geniş şəkildə tətbiq edilmişdir.

\- Müasir C# sintaksisi (`target-typed new`, expression-bodied members, nullable reference types və s.) istifadə olunur.

\- Layihə Microsoft SQL Server üzərində işləyir.

\- Konfiqurasiya `appsettings.json` vasitəsilə idarə olunur.

\- Excel ixracı üçün EPPlus kitabxanasından istifadə edilir.

\- Material Design üslubunda istifadəçi interfeysi üçün MaterialSkin 2 inteqrasiya edilmişdir.
